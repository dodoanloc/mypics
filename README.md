# Agribank Thọ Xuân - Nộp Kết Quả Thi

Hệ thống quản lý nộp ảnh kết quả thi cho nhân viên Agribank Chi nhánh Thọ Xuân.

## Tính năng

- 📸 Người dùng có thể nộp ảnh chụp kết quả thi
- 🗓️ Quản trị viên tạo sự kiện với ngày hết hạn
- 📊 Theo dõi tiến độ nộp ảnh
- 📥 Tải xuống ảnh hàng loạt

## Hướng dẫn sử dụng

### Cho người dùng:
1. Chọn sự kiện muốn nộp ảnh
2. Chọn ảnh chụp kết quả thi (JPG/PNG)
3. Nhập họ tên và số CMND/CCCD
4. Nhấn "Gửi Ảnh"

### Cho quản trị viên:
1. Truy cập phần "Quản Trị"
2. Đăng nhập với tài khoản admin
3. Tạo sự kiện mới hoặc xem tiến độ
4. Tải xuống tất cả ảnh đã nộp

## Thiết kế
- Giao diện đơn giản, màu đỏ/trắng (màu Agribank)
- Font: Plus Jakarta Sans
- Responsive cho mọi thiết bị
